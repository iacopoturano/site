Programming

I've started programming in C while I was in middle school, my parents bought me an Elegoo pack with an Arduino Uno inside. I've messed around with it for almost a year, then Electronics school began. First year of it was pretty much all about soldering, so no programming. Second year was when C seriously kicked in. We used a big Arduino Kit to learn how to program a microcontroller, we also learned to develop console applications. Third year we (thankfully) abbandoned Arduino in favor of the PIC microcontroller family from Microchip. We worked during the entire year on a custom development board with a PIC18F45k22, writing more complex C code. Also learned to create custom printed circuit boards using Autodesk EAGLE and a PCB printer. The last year we moved on C++ (unfortunatelly) and started developing an UI for said development board.

Known programming languages: PIC18 Assembly, C, C++, Ruby, now learning JavaScript, HTML, CSS. My favorite language is C. I love microcontrollers.
